Strings

What is a String

  • A string is text in Python.
  • It is written inside single quotes (' ') or double quotes (" ").
    • text = "Hello"
    text = 'Hello'

Access Characters (Indexing)

  • You can access characters using their index number.
    • text = "Hello"
    print(text[0])

    # H

  • Index starts from 0.

String Slicing

  • Used to get part of a string.
    • text = "Hello World"
    print(text[0:5])

    # Hello

  • start:end
  • End position is not included.

String Methods

  • Common functions used with strings.
  • Examples:
    • text.upper()

    # HELLO


    text.lower()

    # hello


    text.strip()

    # remove spaces


    text.replace("H","J")
    text.split(",")

Concatenation (Joining Strings)

    a = "Hello"
    b = "World"

    print(a + " " + b)
  • Output:
    • Hello World

String Formatting

  • Used to insert variables inside text.
  • Using f-strings:
    • name = "Ali"
    age = 20

    print(f"My name is {name} and I am {age}")

Escape Characters

  • Used for special characters.
  • Examples:
    • \"

    # double quote


    \'

    # single quote


    \n

    # new line


    \t

    # tab


    \\

    # backslash

  • Example:
    • print("He said \"Hello\"")
