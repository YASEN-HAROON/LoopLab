Operators

What Are Operators?

  • Operators are symbols that perform operations on values or variables (called operands).
    They are essential for doing math, comparing values, combining conditions, and more.

Arithmetic Operators

  • Used for mathematical operations :
    • Operator     Meaning
    • +   Addition
    • -   Subtraction
    • *   Multiplication
    • /   Division (float result)
    • //  Floor division (integer result)
    • %   Modulus (remainder)
    • **  Exponentiation (power)
  • Example :
    • 5 + 3   

    # 8


    10 % 3  

    # 1


    2 ** 3  

    # 8

Assignment Operators

  • Used to assign values to variables, including shortcuts :
    • Operator     Meaning
    • =   Assign
    • +=   Add and assign
    • -=   Subtract and assign
    • *=   Multiply and assign
    • /=  Divide and assign
    • //=   Floor divide and assign
    • %=  Modulus and assign
    • **  Power and assign
  • Example:
    • x = 5
    x += 2  

    # same as x = x + 2

Comparison Operators

  • Used to compare values, and return True or False :
    • Operator     Meaning
    • ==  Equal to
    • !=  Not equal to
    • >   Greater than
    • <   Less than
    • >=  Greater or equal
    • <=  Less or equal
  • Example :
    • 5 > 3   

    # True


    5 == 2  

    # False

Logical Operators

  • Used to combine Boolean expressions :
    • Operator     Meaning
    • and   True if both are True
    • or    True if at least one is True
    • not   Reverses True ↔ False
  • Example :
    • True and False   

    # False


    not (5 > 3)      

    # False

Identity Operators

  • Check if two variables refer to the same object (not just equal value) :
    • Operator     Meaning
    • is          True if they are the same object
    • is not    True if different objects
  • Example :
    • a = [1,2,3]
    b = a
    c = [1,2,3]
    a is b     

    # True


    a is c     

    # False

Membership Operators

  • Test whether a value is in a sequence (like list, string, tuple) :
    • Operator     Meaning
    • in           True if value is found
    • not in     True if value is not found
  • Example :
    • 'k' in 'skill'      

    # True


    10 not in [1,2,3]   

    # True

Bitwise Operators

  • Operate on bits of integers :
    • Operator     Meaning
    • &       Bitwise AND
    • `       `
    • ^       Bitwise XOR
    • ~       Bitwise NOT (flip bits)
    • <<      Left shift
    • >>      Right shift
  • Example (conceptual):
    • 5 & 3    

    # 1 (0101 & 0011 → 0001)
