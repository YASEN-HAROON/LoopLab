Sets

What is a Set?

  • A set is a collection that is:
    • Unordered
    • Unchangeable (items cannot be edited)
    • No duplicate values
    my_set = {"apple", "banana", "cherry"}
  • Duplicates are automatically removed

Accessing Set Items

  • You cannot access by index (because sets are unordered)
  • You can:
    • Loop through it
    • Check if an item exists using in
    for x in my_set:
        print(x)

    print("apple" in my_set)

Adding Items

  • Add one item:
    • my_set.add("orange")
  • Add multiple items:
    • my_set.update(["mango", "grape"])

Removing Items

  • Remove item (error if not found) :
    • my_set.remove("banana")
  • Remove item safely :
    • my_set.discard("banana")
  • Remove random item :
    • my_set.pop()
  • Clear set :
    • my_set.clear()

Looping Through a Set

    for item in my_set:
        print(item)

Joining Sets

  • Union (combine sets):
    • set3 = set1.union(set2)

    # or

    set1.update(set2)
  • Intersection (common items):
    • set1.intersection(set2)
  • Difference (items in set1 not in set2):
    • set1.difference(set2)
  • Symmetric Difference (items not common):
    • set1.symmetric_difference(set2)
