Variables

What is a Variable

  • A variable is a container used to store data in a program.
  • Example:
    • x = 5
    name = "John"

Creating Variables

  • In Python, you do not need to declare the type of a variable.
  • A variable is created when you assign a value to it.
  • Example:
    • x = 5
    y = "John"

Changing Variable Type

  • Python allows you to change the type of a variable.
  • Example:
    • x = 4
    x = "Sally"
  • First it was a number, then it became a string.

Variable Naming Rules

  • A variable name must follow these rules:
    • Must start with a letter or _
    • Can contain letters, numbers, and _
    • Cannot start with a number
    • Variable names are case-sensitive
  • Correct examples:
    • age
    user_name
    _myvar
  • Incorrect examples:
    • 2name
    my-name
    my name

Multi-word Variable Names

  • Common styles:
  • snake_case (most common in Python)
    • my_variable_name
  • camelCase
    • myVariableName
  • PascalCase
    • MyVariableName

Assigning Multiple Variables

  • You can assign multiple variables in one line:
    • x, y, z = "Orange", "Banana", "Cherry"
  • Assign the same value:
    • x = y = z = "Orange"

Output Variables

  • Use print() to display variables.
  • Example:
    • x = "Python"
    print(x)
  • Combine text and variables:
    • x = "awesome"
    print("Python is " + x)
  • Multiple variables:
    • x = "Python"
    y = "is"
    z = "awesome"

    print(x, y, z)

Adding Variables

  • Numbers → addition
    • x = 5
    y = 10
    print(x + y)
  • Strings → concatenation
    • x = "Hello "
    y = "World"

    print(x + y)

Global Variables

  • A global variable is created outside a function and can be used inside it.
  • Example:
    • x = "awesome"

    def myfunc():
        print("Python is " + x)

    myfunc()
  • To modify it inside a function, use global.
    • x = "awesome"

    def myfunc():
        global x
        x = "fantastic"

    myfunc()
    print(x)

Deleting Variables

  • You can delete a variable using del.
  • Example:
    • x = 5
    del x
