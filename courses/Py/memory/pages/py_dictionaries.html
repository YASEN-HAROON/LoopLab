Dictionaries

Accessing Items

  • Access value by using the key
  • dict[key] or dict.get(key) (safer)

  • Example :
    • print(student["name"])
    print(student.get("age"))

Changing Values

  • Assign a new value to an existing key
    • student["age"] = 22

Adding Items

  • You can add new key–value pairs :
    • student["city"] = "Cairo"

Removing Items

  • Several ways to remove :
    • Method             What it does
    • pop(key)           Removes that key
    • popitem()         Removes last inserted pair
    • del dict[key]     Deletes by key
    • clear()               Empties entire dict
    student.pop("major")

Looping Through a Dictionary

  • You can loop over :
    • Keys :
      for key in dict:
    • Values :
      for val in dict.values():
    • Key-Value pairs :
      for key, val in dict.items():
  • Example :
    • for key, val in student.items():
        print(key, val)

Copying Dictionaries

  • Shallow copy using copy() method
  • Or use dict() to make a new copy
    • new_dict = student.copy()

Nested Dictionaries

  • A dictionary can contain another dictionary
  • Example :
    • students = {

        "s1": {
            "name": "Ali",
            "age": 20
        },

        "s2": {
            "name": "Sara",
            "age": 22
        }
    }
