Lists

Python Lists – Key Concepts

  • What a List Is
    • A list stores multiple items in one variable using square brackets [].
    • Example :
    fruits = ["apple", "banana", "cherry"]
  • Lists are :
    • Ordered – items stay in the same sequence.
    • Mutable – you can change items after creation.
    • Allow duplicates – the same value can appear more than once.

Basic List Operations

  • Access Items (Indexing)
    • Use indexes to get items :
    fruits[0] → first item
    fruits[-1] → last item

Change List Items

  • Assign a new value to change :
    • fruits[1] = "orange"

Adding Items

  • append() → add an item at the end.
  • insert() → add item at a specific position.
  • extend() → add all items from another list (like joining).

Removing Items

  • remove() → delete a specific value.
  • pop() → remove an item by index (and return it).
  • clear() → remove all items.

Looping Through Lists

  • You can use a for loop to go through every item :
    • for item in fruits:
        print(item)
  • This prints each element one by one.

List Comprehension

  • A fast, concise way to create a new list from an existing one :
    • newlist = [x for x in fruits if "a" in x]
  • This finds all items containing "a" and makes a new list.

Sorting Lists

  • sort() → rearranges items in ascending order by default.
  • reverse() → flips the list order.

Copying Lists

  • Use copy() to make a new list that is separate from the original.

Joining Lists

  • You can join two lists in different ways :
  • Using + :
    • combined = list1 + list2
  • Using extend().
  • Or with list comprehension.
