Get Start

Python Install

  • Many PCs and Macs will have python already installed.
  • To check if python is installed on your computer, open the command line and type :
    • python --version
  • If Python is installed, it will show the Python version.

Python Quickstart

  • Python is an interpreted programming language, this means that as a developer you write Python (.py) files in a text editor and then put those files into the python interpreter to be executed.
  • The way to run a python file is like this on the command line :
    • python helloworld.py
  • Where helloworld.py is the name of your python file.

Your First Python Program

  • Create a file called helloworld.py, and write the following code in it:
    • print("Hello, World!")
  • Save the file and run it in the command line.

  • Output:
    • Hello, World!

The Python Command Line

  • To test a short amount of code in python sometimes it is quickest and easiest to not write the code in a file.
  • This is made possible because Python can be run as a command line itself.
  • Type the following on the Windows, Mac or Linux command line:
    • python

  • After that you can write python code directly:
    • print("Hello, World!")

  • To exit the Python command line interface type:
    • exit()
