Comments

What is a Comment in Python

  • A comment is text in the code that is used to explain the code.
  • Python ignores comments, so they are not executed when the program runs.

Comments are used to:

  • Explain the code
  • Make the code easier to understand
  • Prevent code from running when testing

Single-line Comments

  • In Python, comments start with the # symbol.
  • Everything after # on that line is treated as a comment.
  • Example:

    • # This is a comment

    print("Hello, World!")

Comments at the End of a Line

  • You can also write a comment after a line of code.
  • Example:

    • print("Hello, World!")

    # This is a comment

Using Comments to Disable Code

  • You can use # to stop a line of code from running.
  • Example:

    • # print("Hello, World!")

    print("Cheers, Mate!")

Multi-line Comments

  • Python does not have a special syntax for multi-line comments.
  • However, there are two common ways:

    • 1. Using # on each line

    # This is a comment
    # written in
    # multiple lines

    2. Using triple quotes

    """
    This is a comment
    written in
    multiple lines
    """

  • If this text is not assigned to a variable, Python ignores it.
