Tuples

What Are Python Tuples

  • Tuples are like lists but ordered and immutable (meaning you cannot change items after creation)
  • Tuples are written with parentheses ( )
  • They can contain duplicate items and multiple data types

Accessing Tuple Items

  • Access items using index numbers in square brackets (e.g., mytuple[0])
  • Python supports negative indexing (-1 is last item)
  • You can slice a tuple to get a range of items
  • Check membership with in keyword (e.g., "apple" in mytuple)

Updating Tuples (Workarounds)

  • Because tuples are immutable, you can’t change items directly
  • Workaround: convert tuple to a list, modify it, then convert back to a tuple

Unpacking Tuples

  • Tuple unpacking lets you assign each tuple item to a variable in one statement
    • a, b, c = ("apple", "banana", "cherry")
  • If there are more values than variables, use * to collect the remaining items

Looping Through Tuples

  • You can loop with a for loop, going through each item directly
  • Or loop by index using range(len(tuple))
  • while loops also work with a counter index

Joining Tuples

  • Use the + operator to combine two tuples into a new one
  • You can also multiply a tuple by a number to repeat its items

Tuple Methods

  • Python tuples have two built-in methods :
    • count() — returns how many times a value appears
    index() — returns the position of a value
