Syntax

Python Syntax – Important Points

  • Python syntax refers to the rules that define how Python code must be written so the interpreter can understand it.

Running Python

  • Python code can be executed in two main ways:
  • Using the Python command line (interactive mode) where you write code and run it immediately.
  • Writing code inside a { .py } file and running the file.

Indentation

  • One of the most important features of Python syntax is indentation.
  • Python uses indentation to define blocks of code (for example inside loops, functions, and conditions).
  • Other languages use { }, but Python uses spaces.
  • The common standard is 4 spaces.
  • Example:
    • if 5 > 2:
        print("Five is greater than two")
  • If indentation is incorrect, Python will produce an Indentation Error.

Variables

  • Variables in Python are created when you assign a value to them.
  • You do not need to declare the variable type before using it.
  • Example:
  • Python automatically detects the data type.

Comments

  • Comments are used to explain the code and make it easier to understand.
  • A comment starts with #.
  • Python ignores comments when executing code.
  • Example:
    • # This prints a message
    print("Hello World")
  • Comments are useful for documentation and for temporarily disabling code.

Python Statements – Important Points

  • A statement is a single instruction that Python executes.
  • Programs are made of many statements that run in order from top to bottom.
  • Example:
    • print("Hello")
    print("Welcome to Python")
    print("Learning programming")
  • Each line above is a separate statement.

End of Statements

  • In Python, a statement usually ends when the line ends.
  • Unlike languages such as C, Java, or JavaScript, Python does not require semicolons (;).
  • Example:
    • print("Hello World")

Multiple Statements on One Line

  • Python allows writing multiple statements on the same line using a semicolon ;, but it is not recommended because it reduces readability.
  • Example:
    • print("Hello"); print("World")
  • Good Python style is to write one statement per line.
