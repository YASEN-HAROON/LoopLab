Data Types

What is a Data Type?

  • A data type defines the kind of value a variable can hold (text, number, list, etc.).
  • Different types support different operations.
  • Example:
    • x = 5
    print(type(x))

    # Output: class 'int'

  • type() tells you the type of a variable.

Python Data Types

  1. Text Type
    • str → stores text.
    • name = "John"
  2. Numeric Types
    • Numbers:
      • int → integer
      • float → decimal number
      • complex → complex number
      • Example:
      x = 10      

      # int


      y = 3.5     

      # float


      z = 1j      

      # complex


  3. Sequence Types
    • Stores multiple items in order:
    • list → mutable list
    • tuple → immutable list
    • range → sequence of numbers
    • Example:
      • fruits = ["apple", "banana", "cherry"]  

      # list


      numbers = (1, 2, 3)                     

      # tuple


      r = range(6)
  4. Mapping Type
    • dict → stores key-value pairs
      • person = {"name":"Ali", "age":20}
  5. Set Types
    • set
    • frozenset
    • Stores unique elements only (no duplicates).
    • Example:
      • s = {"apple", "banana", "cherry"}
  6. Boolean Type
    • bool → True or False
      • x = True
  7. Binary Types
    • For binary data:
      • bytes
      • bytearray
      • memoryview
  8. None Type
    • NoneType → represents no value
      • x = None
