Booleans

What is a Boolean

  • A Boolean is a data type in Python.
  • It has two values only:
    • True
    • False
  • Example:
    • x = True
    y = False
    print(x)
    print(y)

Comparisons Return Boolean

  • When you compare values, Python returns True or False.
  • Example:
    • print(10 > 9)

    # True


    print(10 == 9)

    # False


    print(10 < 9)

    # False

Using Boolean in if Statements

  • Booleans are used in conditions to make decisions.
  • Example:
    • a = 200
    b = 33

    if b > a:
        print("b is greater than a")
    else:
        print("b is not greater than a")

The bool() Function

  • The bool() function converts a value into True or False.
  • Example:
    • print(bool("Hello"))

    # True


    print(bool(15))

    # True


Values That Are True

  • Most values are True if they contain something, such as:
  • Non-empty string "Hello"
  • Non-zero number 10
  • Non-empty list [1,2,3]

Values That Are False

  • Some values are always False:
    • False
    None
    0
    ""
    []
    {}
    ()
